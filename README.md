# Andromeda Eggs ===> Based on Penguins Eggs
Tool used in andromeda linux ===> based on opensuse

### Installation

Clone Repo 
```bash
git clone https://github.com/Andromeda-Linux/Andromeda-Eggs.git
```

Go to file
```bash
cd Andromeda-Eggs
```

Add Permission
```bash
chmod a+x install.sh
```

Run Install.sh
```bash
install.sh
```

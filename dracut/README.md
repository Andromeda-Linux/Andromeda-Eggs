# dracut on Fedora

## Install

### fedora 
```
sudo dnf -y install \
	dracut \
	dracut-live
```

### opensuse
```
sudo zypper -y install \
	dracut \
	dracut-tools
```

### pacchetti da installare/rimuovere per opensuse
sudo zypper rm multipath-tools
sudo zypper install libxkbcommon-tools util-linux kf6-qqc2-desktop-styl
